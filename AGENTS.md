# React 19

## Overview

This project is built with **React 19**. All React code, patterns, and recommendations should follow the React 19 API and best practices.

## Guidelines

- Prefer React 19 APIs over deprecated patterns.
- Do not suggest APIs that were removed or deprecated in React 19.
- Follow the official recommendations for hooks, rendering, forms, and server/client components.
- When unsure about React 19 behavior or a newly introduced API, consult the official documentation.
